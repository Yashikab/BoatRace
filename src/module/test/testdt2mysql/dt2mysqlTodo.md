## 開催会場のデータ

- テーブル名: holdjyo_tb
- カラム
  - datejyo_id: yyyymmdd{jyo_code: %2d} primary key
  - holddate: yyyymmdd
  - jyo_cd: 会場コード
  - jyo_name: 会場名

- [x] クラスの作成
